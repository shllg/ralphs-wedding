# Wedding Invitation Manager

A quick interview test application for managing wedding events and invitations.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Supabase (PostgreSQL)
- Prisma 7
- Tailwind CSS 4
- TypeScript
